Privacy Policy

Last Updated: January 6, 2026

"NHI Cloud Analyzer 2.0" (hereinafter referred to as "this Extension") respects and protects user privacy. This policy explains how we handle your data.

1. Data Collection and Storage

1.1 Locally Stored Data

This Extension uses Chrome Storage API to store the following data locally in your browser:

  • User preferences (display options, tracking days, etc.)
  • AI service API Keys (if you choose to use the GAI feature)
  • GAI conversation history (stored locally only, not synced to cloud)
Important: Your API Keys are stored only in your local browser and are never transmitted to any third-party servers (except the AI service provider you choose).

1.2 Data We Do NOT Collect

This Extension does NOT:

  • Collect or transmit any data to the Extension developers
  • Track your browsing behavior
  • Store patient data on any external servers
  • Share your data with advertisers or data brokers

2. GAI Smart Analysis Feature

Note: The GAI feature is optional and disabled by default. Please read the following before enabling.

2.1 Data Transmission

When you enable and use the GAI Smart Analysis feature, the system will transmit the medical data types you select to your chosen AI service provider for analysis:

AI Service Provider API Endpoint Privacy Policy
Cerebras (Recommended) api.cerebras.ai Link
Groq api.groq.com Link
Google Gemini generativelanguage.googleapis.com Link
OpenAI api.openai.com Link

2.2 Automatic De-identification Protection

Privacy Protection Mechanism: This Extension includes an automatic de-identification system that masks the following sensitive information before transmitting imaging reports and other data to AI services:
  • Patient Information: Names, medical record numbers, bed numbers
  • Medical Staff Information: Physician names, radiologist names, nurse names, various medical license numbers
  • System IDs: Order numbers, image numbers, endoscopy numbers

Supports deep optimization for multiple hospital report formats.

2.3 Selective Data Transmission

You can freely choose which types of medical data to transmit for analysis (10 types available). Unselected data types will not be transmitted.

2.4 Free Tier Warning

Important Warning: Some AI services' free tiers may use your submitted content to improve their AI models. If you handle highly sensitive medical data, we recommend:
  • Using paid plans (typically not used for model training)
  • Avoiding transmission of directly personally identifiable information
  • Reading each AI service provider's data usage policy

3. Permission Explanations

This Extension requests the following permissions:

Permission Purpose
webRequest Intercept NHI cloud system API responses to extract medical data
storage Store user settings and API Keys locally
scripting Inject content scripts into NHI cloud webpages
clipboardWrite Support copy functionality
host_permissions Access NHI cloud website and AI API endpoints

4. Data Control Rights

You can at any time:

  • Disable GAI Feature: Turn off "Enable GAI Smart Analysis Sidebar" in settings
  • Delete API Keys: Clear saved API Keys in settings
  • Clear Conversation History: Use the "Clear History" button in the Chat tab
  • Reset All Settings: Use "Reset to Defaults" or remove the extension

5. Data Security

  • This Extension only operates on the medcloud2.nhi.gov.tw domain
  • All local data uses Chrome's secure storage mechanism
  • All API communications use HTTPS encrypted transmission
  • This Extension is open source and the code is available for review

6. Third-Party Services

This Extension may connect to the following third-party services:

  • drugtw.com: Drug image lookup (no patient data transmitted)
  • AI Service Providers: Only when you enable GAI feature and actively trigger analysis

7. Policy Updates

This privacy policy may be updated from time to time. Significant changes will be announced through extension update notes or the GitHub project page.
